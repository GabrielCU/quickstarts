== Installation

=== Download required software

. link:https://downloads.teradata.com/download/database/teradata-express-for-vmware-player[Teradata Express for VMware Player]. If you have not used the Teradata downloads website before, you will need to link:https://downloads.teradata.com/user/register[register].
. link:https://www.vmware.com/products/workstation-player.html[VMware Player].
+
IMPORTANT: VMware doesn't offer VMware Player for MacOS. If you are on a Mac, you will need to install link:https://www.vmware.com/products/fusion/fusion-evaluation.html[VMware Fusion] instead. It's a paid product but VMware offers a free 30-day trial.

=== Run installers

Install VMware Player or VMware Fusion by running the installer and accepting the default values.

=== Run Teradata Vantage Express

. Go to the directory where you downloaded Teradata Vantage Express and unzip the downloaded file.
. Double-click on the `.vmx` file. This will start the VM image in VMware Player/Fusion.
